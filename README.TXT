PROJECT TITLE: Text Based RP Game (AP CSA CHRHS)

PURPOSE OF PROJECT: To explore further opportunities within the Java programming language. This project is designed to provide a fun and engaging way to learn and apply Java concepts through a text-based role-playing game.

VERSION or DATE: v0.1 05/30/2024

HOW TO START THIS PROJECT:
1. Open the project in your preferred Java IDE (e.g., BlueJ, IntelliJ IDEA, Eclipse).
2. Locate the `Game` class in the project files.
3. Run the `Game` class to start the game.

AUTHORS: Evan-CHRHS 2025

USER INSTRUCTIONS: 
- Use text commands to interact with the game world.
- Common commands include:
  - `go [direction]` to move (e.g., `go north`).
  - `take [item]` to pick up an item (e.g., `take flashlight`).
  - `drop [item]` to drop an item (e.g., `drop flashlight`).
  - `inventory` to view your current items.
  - `look` to look around the room.
  - `attack [target] with [weapon]` to attack a target with a specified weapon (e.g., `attack zombie with gun`).
  - `consume [item]` to consume a food item (e.g., `consume apple`).
  - `help` to display the list of commands.
  - `quit` to exit the game.

Enjoy your adventure in the text-based RPG!
